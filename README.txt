This is a guided project from a PostgreSQL specialization by Dr Charles Severance from the University of Michigan.

------elasticbook.py

Accepts a book in txt format, slices the text into paragraphs, and stores it in ElasticSearch.

Add database connection parameters in hidden-dist.py, and run the elasticbook.py.

------elasticmail.py

Pulls email data from the web and stores different parts of the mail in ElasticSearch.

Add database connection parameters in hidden-dist.py, and run the elasticmail.py.
